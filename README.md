O Next é um framework react para produção.

Seu principal objetivo é adicionar SEO(Search Engine Optimization) o next possuí também um série de vantagens.


Pages -> São todas as rotas da minha aplicação não é necessário adicionar módulos de terceiros, ele mesmo cria as proprías rotas baseadas nas pages.

Podemos buscar ids especificos de determinada informação baseado no [id] informados.
Exemplo queremos acessar a rotas de alunos e buscar o aluno que possui o id 3.

Folder Alunos/[id].js -> e o next entendera que isso é um recurso dinamico.

Com o next criamos um folder de alunos isso ira criar rotas especificas para alunos, caso se quisermos acessar um rota com o mesmo nome do folder basta adicionar um index no folder.
